# Fraction Quiz
This program can be used to improve knowledge of fractions.


## Run
Firstly install dependencies:
```
pip install -r requirements.txt
```

After that run [main.pyw](main.pyw) file, program consists of 4 topics, each topic consists from 3 to 4 exercises.

## Tests
To run tests use command:

```
python -m unittest discover
```

## Build
Nuitka==0.6.17.7 is used for building, you can run [build.bat](build.bat) script.

## Dependencies
[unicodeit==0.7.5](https://pypi.org/project/unicodeit/0.7.5/)  and built-in module [tkinter](https://docs.python.org/3/library/tkinter.html).
